# DataStructures

DataStructures Repository for Self Studying the Data Structures
(Textbook used: DataStructures by Koffman & Wolfgang)

So far implemented:
  - ArrayList
  - Single-Linked List
  - Double-Linked List
  - Linked-Stack: Postfix Evaluator, Infix Evaluator, Infix Compiler
  - Single-Linked Queue
  - Circular Array Queue
